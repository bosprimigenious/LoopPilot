## 阶段 A — 负例显式 BLOCKED

**目标**：fail-closed；禁止 unknown adapter 静默 mock 回退。

| 项 | 实现 |
|----|------|
| `fake_adapter` | `AdapterBlockedError` → outcome **BLOCKED** |
| `unknown_adapter` | 同上，`unknown adapter: {id}` |
| 缺 API key | `validate_adapter_run` → BLOCKED（已有，测试加强） |
| 默认 real disabled | `--adapter cursor_cli` 无 opt-in → BLOCKED |

**测试**：`test_unknown_adapter_is_blocked`, `test_fake_adapter_is_blocked`, `test_missing_api_key_is_blocked`, `test_real_adapter_disabled_by_default`

**状态**：✅ 0.3.0a1
