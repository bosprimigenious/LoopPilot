# 12 0.4-b 任务入口（inbox / queue / today）

本文说明 **0.4-b** 个人任务入口 CLI。英文验收清单见 [development/44-personal-daily-loop-0.4b-acceptance.md](../development/44-personal-daily-loop-0.4b-acceptance.md)。

## 前置条件

- **0.4-a 已完成**：SQLite migration、db CLI、recovery-scan
- 配置需启用 sqlite：

```yaml
runtime:
  state_backend: sqlite
  sqlite_path: var/state/loop_pilot.db
  lock_dir: var/locks
```

默认仓库 `config/loop-pilot.yaml` 仍为 `json`（0.1 基线）；0.4-b 命令仅在 sqlite 后端可用。

## 命令概览

| 命令 | 说明 |
|------|------|
| `loop-pilot inbox add <text> [--source manual\|daily-news] [--loop intern\|paper]` | 写入 Inbox |
| `loop-pilot inbox list [--status open\|all]` | 列出 Inbox |
| `loop-pilot inbox archive <id>` | 归档 |
| `loop-pilot inbox import-daily-news --from <candidate-actions.json>` | 从 DailyNews 产物导入 |
| `loop-pilot queue promote <inbox-id> [--loop …] [--priority N]` | Inbox → Queue |
| `loop-pilot queue list` | 列出队列 |
| `loop-pilot queue demote <queue-id>` | Queue → Inbox |
| `loop-pilot today` | 今日焦点（`scheduled_for` 为当天的 queue 项） |
| `loop-pilot today add <inbox-id>` | 快捷 promote + 排入今日 |

## 数据流

```text
Anything → Inbox → Queue → Today（queue.scheduled_for 视图）
                ↑
         DailyNews candidate-actions（import-daily-news）
```

**Today 实现**：无独立 `today_items` 表；通过 `queue_items.scheduled_for` 标记当日焦点。
