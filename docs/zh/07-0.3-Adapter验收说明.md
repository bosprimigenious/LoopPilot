## 0.3 证明什么

0.3 **不等于**旧称「V1」。本阶段只证明：

- Mock 之外的真实 Adapter（Cursor CLI、OpenAI-compatible）能在**严格门禁**下接入
- ModelRouter、ToolBroker、默认 `allow_real_adapters=false` 生效
- 0.1 / 0.2 回归不退化

**不属于 0.3**：SQLite 恢复、inbox/queue/today、审批 CLI、调度、PyPI、团队功能（见 [08-0.4-个人日用闭环.md](08-0.4-个人日用闭环.md)）。

## 常用命令

```bash
# 日常回归（无需真实 Adapter）
pytest -q
loop-pilot doctor
loop-pilot adapters list
loop-pilot adapters doctor
loop-pilot run all --fixture-set mini --dry-run
loop-pilot run all --profile demo --dry-run

# 验证默认拦截（应 blocked）
loop-pilot run intern --workspace examples/intern_demo --adapter cursor_cli --dry-run
loop-pilot run intern --adapter fake_adapter --dry-run   # unknown → blocked

# CI 同款自动化
python scripts/verify_0_3_acceptance.py

# 真实 run（Full DoD；需 config + env + 显式 flag）
loop-pilot run intern --workspace examples/intern_demo --allow-write --allow-real-adapters
loop-pilot run paper --workspace examples/paper_demo --allow-write --allow-real-adapters
```
