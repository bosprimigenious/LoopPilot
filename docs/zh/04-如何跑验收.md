# 04 如何跑验收

从仓库根目录执行。先 `pip install -e ".[dev]"`。

## 通用基线（每次 PR / 分支切换）

```bash
loop-pilot doctor
pytest -q
ruff check .
```

## 0.1 Mini-MVP 验收

```bash
loop-pilot run intern --fixture simple_python_bug --dry-run
loop-pilot run paper --fixture unsupported_claim --dry-run
loop-pilot run daily-news --fixture github_star_snapshots --dry-run
loop-pilot run all --fixture-set mini --dry-run
loop-pilot status
```

**通过标准：** 三条 Loop 走到 Report；`allow_real_adapters=false`；无假 `resume` 命令。

清单：[development/32-mini-mvp-acceptance.md](../development/32-mini-mvp-acceptance.md)

## 0.2 Practical MVP 验收

```bash
loop-pilot run intern --workspace examples/intern_demo --dry-run
loop-pilot run paper --workspace examples/paper_demo --dry-run
loop-pilot run daily-news --source-profile demo --dry-run
loop-pilot run all --profile demo --dry-run
```

**通过标准：** 受控 workspace 可观察、诊断、出报告；0.1 fixture 仍全绿。

清单：[development/35-practical-mvp-0.2-acceptance.md](../development/35-practical-mvp-0.2-acceptance.md)
**CI：** GitHub Actions 在「Practical MVP demo dry-run runs」步骤中执行上述 0.2 命令（含 `run all --profile demo --dry-run`）。


## 0.3 Adapter MVP 验收

**门禁（默认应拦截）：**

```bash
loop-pilot adapters list
loop-pilot adapters doctor
loop-pilot run intern --workspace examples/intern_demo --adapter cursor_cli --dry-run
# expect: BLOCKED
```

**回归（必须保持）：**

```bash
pytest -q
loop-pilot run all --fixture-set mini --dry-run
loop-pilot run all --profile demo --dry-run
```

**启用真实 Adapter（手动，非 CI）：**

1. `config/loop-pilot.yaml` → `runtime.allow_real_adapters: true`
2. 配置 `config/adapters.yaml` 与 API key / CLI
3. `loop-pilot adapters doctor`
4. 加 `--allow-real-adapters` 再跑

清单：[development/36-adapter-mvp-0.3-acceptance.md](../development/36-adapter-mvp-0.3-acceptance.md)  
安全策略：[development/37-adapter-safety-policy.md](../development/37-adapter-safety-policy.md)

## 0.4+ 命令（当前 JSON 后端会拒绝）

以下需 `runtime.state_backend: sqlite`（0.4 实现）：

```bash
loop-pilot resume <run-id>
loop-pilot approve <run-id>
loop-pilot reject <run-id> --reason "..."
```

Mini/0.3 JSON 后端按设计拒绝这些命令。
