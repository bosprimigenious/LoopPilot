# 00 快速开始

## 这是什么

LoopPilot 是你本机上的 **AI 工作闭环运行时**：每天三条约 30 分钟的 Loop，产出可读的 Markdown 报告，而不是一堆聊天日志。

## 安装

```bash
git clone https://github.com/bosprimigenious/LoopPilot.git
cd LoopPilot
pip install -e ".[dev]"
python scripts/bootstrap_local.py   # 可选
loop-pilot doctor
```

`doctor` 应报告配置可读、MockAdapter 可用。

## 30 秒跑通

**0.2 demo workspace（推荐首次体验）：**

```bash
loop-pilot run intern --workspace examples/intern_demo --dry-run
loop-pilot run all --profile demo --dry-run
loop-pilot status
```

**0.1 fixture 回归：**

```bash
loop-pilot run intern --fixture simple_python_bug --dry-run
loop-pilot run all --fixture-set mini --dry-run
```

**测试：**

```bash
pytest -q
```

## 常用 CLI

| 命令 | 作用 |
|------|------|
| `loop-pilot doctor` | 环境与健康检查 |
| `loop-pilot run <loop> ...` | 跑一次 Loop |
| `loop-pilot run all ...` | 顺序跑三条 |
| `loop-pilot status` | 最近运行摘要 |
| `loop-pilot inspect <run-id>` | 查看单次运行 |
| `loop-pilot adapters list` | 0.3：列出 Adapter |
| `loop-pilot adapters doctor` | 0.3：Adapter 门禁检查 |

## 默认安全

- `runtime.allow_real_adapters: false` — 只用 MockAdapter
- `--dry-run` — 不执行真实写入（除非显式 `--allow-write`）
- 真实 Cursor CLI / API 需改配置 **且** 加 `--allow-real-adapters`
